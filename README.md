# Cocktail Menu

A full-stack application designed to host cocktail recipes and
images, enabling users to easily search, sort, and filter content to
find what they'd like to drink.

<img src="https://raw.githubusercontent.com/jacksoncutler/jacksoncutler/refs/heads/main/assets/images/screenshot-cocktail-site.jpg" style="width: 400px;">

Live demo can be found [here](https://cocktail-menu-fe2f8.web.app).

## Quick Start

Clone repo and navigate into root directory.

### Install Dependencies

`npm install`

### Run Locally

`npm run start`
